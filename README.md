# Plural 

Возвращает строку с множественным числом

## Установка
```
pip install pluralru
```

## Как использовать Plural
```python
from pluralru.pluralru import * #plural_seconds, plural_minutes, plural_hours, plural_days, plural_months, plural_years

s = plural_seconds(15)
print(s) # 15 секунд
```
